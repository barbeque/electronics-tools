There's a bunch of tools here I needed for one project or another.

None of them are probably "complete," but maybe they'll help you in a pinch.

## ines.py
A very partial iNES header decoder. Knows only about a few mappers.

## esr.py
Checks a recorded electrolytic capacitor's ESR and compares it against a known table of "worst case" ESR. Tells you if the cap is bad.
